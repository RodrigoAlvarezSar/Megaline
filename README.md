# Megaline
Project Description

Study the usage users have of the Megaline’s different packages, Surf and Ultimate in the different services that are offered, phone calls, messages and data, through the different months of the year, as well as how profitable are them to the company.

Tools and Libraries

Pyhon Pandas
Pyhon Matplotlib
Python Numpy
Pyhon Scipy
Python Seaborn


Versión en Español
